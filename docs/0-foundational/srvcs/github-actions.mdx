

### Enabling Environment Variables 

> Passing Environment Variables from gh secrets to gh acction and using it as process.env. in gh pages

|GitHub Secret|GitHub Actions|GitHub Actions|
|---|---|---|
|TYPESENSE_SEARCH_API_KEY|TS_SEARCH_API_KEY|process.env.TS_SEARCH_API_KEY|
|TYPESENSE_SCRAPER_API_KEY| used only in GH actions|used only in GH actions|
|TYPESENSE_INSTANCE_URL_01|TS_INSTANCE_URL_01|process.env.TS_INSTANCE_URL_01|
|TYPESENSE_COLLECTION_NAME|TS_COLLECTION_NAME|process.env.TS_COLLECTION_NAME|
